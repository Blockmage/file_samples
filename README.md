# file_samples

IoCs - Indicators of Compromise, i.e., collected samples of malicious code. Primarily for files which are too large to
be shared as Gists. Where possible, both the obfuscated and deobfuscated forms of the files are provided.

> [!WARNING]
> These files contain samples of malicious code (Indicators of Compromise). Handle with extreme caution. Do not execute,
> open, or interact with them outside of a secure, isolated environment to avoid potential harm to your system or data.
